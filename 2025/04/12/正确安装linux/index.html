正确安装linux

四月 12, 2025

一、准备工作

  1. 选择合适的 Linux 发行版
    新手推荐：Ubuntu（桌面友好）、Linux Mint（基于 Ubuntu，更接近 Windows 操作习惯）、Deepin（国产，美观易用）。
    技术向 / 服务器：Debian（稳定）、Fedora（最新技术）、CentOS（企业级，适合服务器）。
    下载 ISO 镜像：从官方网站下载对应版本（如 Ubuntu 官网），确保选择 64 位版本（主流配置）。
  2. 准备启动盘
    工具：
    Windows：Rufus 或 BalenaEtcher（推荐，简单易用）。
    macOS/Linux：使用终端命令 dd 或图形工具 BalenaEtcher。
    步骤：
    插入 8GB 以上 U 盘（注意备份数据，制作启动盘会格式化 U 盘）。
    打开工具，选择下载的 ISO 文件和 U 盘设备，点击 “开始” 制作启动盘。

二、安装前的准备

  1. 备份重要数据
    全新安装或双系统分区可能导致数据丢失，建议提前备份文件到外部存储。
  2. 进入 BIOS/UEFI 设置
    启动电脑时按特定按键（不同品牌不同，常见：F2、Delete、F12、Esc），进入 BIOS/UEFI。
    设置项调整：
    启用 UEFI 模式（推荐，支持大于 2TB 硬盘）。
    关闭 安全启动（Secure Boot，部分发行版需关闭以支持第三方驱动）。
    设置 U 盘为第一启动项（或在启动菜单中选择 U 盘启动）。

三、正式安装（以 Ubuntu 为例）

  1. 从 U 盘启动
    电脑启动后，选择 Install Ubuntu（或类似选项），进入安装向导。
  2. 选择语言和安装类型
    语言：选择简体中文或英文。
    安装类型：
    全新安装：选择 清除磁盘并安装 Ubuntu（适合单系统，会删除磁盘所有数据）。
    双系统安装：选择 与 Windows 共存（前提是已有 Windows 系统，且已预留空闲磁盘空间）。
    手动分区（高级）：选择 其他选项（适合自定义分区，如服务器或多磁盘用户）。
  3. 分区（关键步骤）
    新手建议自动分区（选择 “清除磁盘” 或 “与 Windows 共存”），系统会自动分配分区。
    手动分区（示例，适用于独立磁盘）：
    ESP 分区（UEFI 模式必需）：300MB - 500MB，格式为 FAT32，用于存储引导文件。
    根分区（/）：建议至少 20GB（日常使用），格式为 ext4。
    home 分区（/home）：可选，用于存储用户数据，格式为 ext4（若希望系统和数据分离）。
    交换分区（swap）：可选，大小建议等于或小于内存容量（如 16GB 内存设 16GB swap），用于虚拟内存。
  4. 设置用户信息
    输入姓名、用户名、密码（建议设置密码，避免空密码），勾选 “自动登录”（可选）。
  5. 开始安装
    等待安装过程（约 10-30 分钟，取决于硬件性能），期间可选择安装更新和第三方软件（如 Chrome、Flash）。
  6. 安装完成，重启系统
    安装完成后，按照提示重启电脑，拔出 U 盘，系统会自动进入新安装的 Linux 系统。

四、双系统安装注意事项

提前在 Windows 中预留空间：
通过 磁盘管理 右键点击 Windows 所在磁盘，选择 “压缩卷”，预留至少 50GB 空闲空间（根据需求调整）。
引导顺序问题：
安装后若默认启动 Windows，需进入 BIOS/UEFI 手动选择 Linux 引导项，或通过工具（如 EasyBCD）修复引导菜单。
分区时避免误删 Windows 分区：
手动分区时注意识别磁盘标签（如 Windows 的 NTFS 分区，通常名为 Windows 或无标签），切勿删除系统分区。

五、安装后配置

更新系统：

安装常用软件：
浏览器（Chrome/Firefox）、办公软件（LibreOffice）、开发工具（VS Code、Git 等）。
配置硬件驱动：
显卡驱动：Nvidia 用户需安装专有驱动（通过系统设置中的 “软件和更新”→“附加驱动”）。
无线网卡 / 蓝牙驱动：部分老旧设备可能需要手动安装，参考硬件厂商文档或社区教程。
设置启动项（双系统）：
若引导菜单异常，可安装 Boot-Repair 修复。

六、常见问题

无法从 U 盘启动：
检查 U 盘是否制作成功（可尝试重新制作），或 BIOS/UEFI 中启动模式是否正确（UEFI 对应 UEFI 启动盘，Legacy 对应传统启动盘）。
分区错误导致数据丢失：
务必提前备份数据，手动分区时仔细确认磁盘和分区信息。
图形界面异常：
尝试以 “安全模式” 启动（安装时选择），或安装正确的显卡驱动。
总结
Linux 安装流程标准化，新手建议选择 Ubuntu 或 Linux Mint 并使用自动分区，避免复杂操作。关键是做好数据备份、选择合适的发行版，并根据硬件和使用场景调整分区和驱动配置。遇到问题时，参考官方文档或社区（如 Ubuntu 论坛、Reddit r/linux4noobs）获取帮助。
